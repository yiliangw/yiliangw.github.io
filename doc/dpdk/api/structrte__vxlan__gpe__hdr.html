Detailed Description

VXLAN-GPE protocol header (draft-ietf-nvo3-vxlan-gpe-05). Contains the 8-bit flag, 8-bit next-protocol, 24-bit VXLAN Network Identifier and Reserved fields (16 bits and 8 bits).
