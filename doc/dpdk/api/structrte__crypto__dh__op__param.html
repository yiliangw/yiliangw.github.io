◆ priv_key

rte_crypto_uint priv_key

Output - generated private key when ke_type is RTE_CRYPTO_ASYM_KE_PRIV_KEY_GENERATE.

Input - private key when ke_type is one of: RTE_CRYPTO_ASYM_KE_PUB_KEY_GENERATE, RTE_CRYPTO_ASYM_KE_SHARED_SECRET_COMPUTE.

In case priv_key.length is 0 and ke_type is set with RTE_CRYPTO_ASYM_KE_PUB_KEY_GENERATE, CSRNG capable device will generate a private key and use it for public key generation.
